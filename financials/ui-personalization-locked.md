# <a name="why-a-page-is-locked-from-personalizing"></a><span data-ttu-id="f9567-103">Waarom is een pagina vergrendeld voor personaliseren?</span><span class="sxs-lookup"><span data-stu-id="f9567-103">Why a Page is Locked From Personalizing</span></span>
<span data-ttu-id="f9567-104">Als er een vergrendelingspictogram is te zien is op de balk **Personaliseren** wanneer u een pagina opent (zoals weergegeven), kunt u op dit moment geen persoonlijke instellingen meer wijzigen op de pagina.</span><span class="sxs-lookup"><span data-stu-id="f9567-104">If there is a lock icon in the **Personalizing** bar when you open a page (as shown), this means that you are currently prevented from making any more personalization changes to the page.</span></span>

<span data-ttu-id="f9567-105">![Personaliseringsvergrendeling](media/personalization-locked.png "Personaliseringsvergrendeling")</span><span class="sxs-lookup"><span data-stu-id="f9567-105">![Personalize Lock](media/personalization-locked.png "Personalize lock")</span></span>

<span data-ttu-id="f9567-106">Dit komt doordat we de manier waarop personalisatie achter de schermen werkt hebben gewijzigd sinds u de pagina voor het laatst hebt gepersonaliseerd.</span><span class="sxs-lookup"><span data-stu-id="f9567-106">This is because we changed the way personalization works behind the scenes since the last time that you personalized the page.</span></span> <span data-ttu-id="f9567-107">De oude en de nieuwe aanpak werken helaas niet goed samen.</span><span class="sxs-lookup"><span data-stu-id="f9567-107">Unfortunately, the old way and new of doing things do not work together.</span></span>

<span data-ttu-id="f9567-108">De pagina bevat momenteel de laatste personalisatiewijzigingen die u hebt aangebracht.</span><span class="sxs-lookup"><span data-stu-id="f9567-108">The page currently includes the last personalization changes that you made.</span></span> <span data-ttu-id="f9567-109">Als u wilt doorgaan met het personaliseren van de pagina, kunt u het vergrendelingspictogram kiezen en vervolgens **Ontgrendelen**.</span><span class="sxs-lookup"><span data-stu-id="f9567-109">If you want to continue personalizing the page, then you can choose the lock icon and then **Unlock**.</span></span> <span data-ttu-id="f9567-110">Als u de pagina ontgrendelt, wordt de huidige personalisatie van de pagina gewist en begint u dus opnieuw.</span><span class="sxs-lookup"><span data-stu-id="f9567-110">Just be aware that if you choose to unlock the page, the current personalization of the page will be cleared, and you will have to start from scratch.</span></span>

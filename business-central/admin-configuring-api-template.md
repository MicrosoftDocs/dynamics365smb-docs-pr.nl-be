## <a name="assign-the-template-to-an-api"></a><span data-ttu-id="ec7bc-113">De sjabloon toewijzen aan een API</span><span class="sxs-lookup"><span data-stu-id="ec7bc-113">Assign the template to an API</span></span>

<span data-ttu-id="ec7bc-117">Als er meer dan één sjabloon is geselecteerd voor een API (Pagina-id), worden de sjablonen toegepast in de volgorde die is gedefinieerd in de kolom **Volgorde** .</span><span class="sxs-lookup"><span data-stu-id="ec7bc-117">If there is more than one template selected for an API (Page ID), the templates are applied in the order defined in the **Order** column.</span></span>   
<span data-ttu-id="ec7bc-118">Wanneer elke sjabloon wordt toegepast, worden veldwaarden die in de sjabloon zijn gedefinieerd, alleen toegepast op velden waarvoor nog geen waarde is gedefinieerd, hetzij expliciet in de sjabloon, hetzij in een eerder toegepaste sjabloon in de volgorde.</span><span class="sxs-lookup"><span data-stu-id="ec7bc-118">When each template is applied, field values defined in the template are only applied to fields that have not already had a value defined, either explicitly in the API, or in a previously applied template in the order.</span></span> 
3. <span data-ttu-id="ec7bc-119">Selecteer een waarde voor **Pagina-id** .</span><span class="sxs-lookup"><span data-stu-id="ec7bc-119">Select a **Page ID** value.</span></span>  
<span data-ttu-id="ec7bc-120">Dit is de pagina voor de API waarop de sjabloon wordt toegepast.</span><span class="sxs-lookup"><span data-stu-id="ec7bc-120">This is the page for the API to which the template will be applied.</span></span> <span data-ttu-id="ec7bc-121">De opzoekactie **Pagina-id** biedt een overzicht van alle API's die beschikbaar zijn in de bibliotheek.</span><span class="sxs-lookup"><span data-stu-id="ec7bc-121">The **Page ID** lookup provides a list of all APIs available in the library.</span></span>
4. <span data-ttu-id="ec7bc-122">Selecteer een waarde in het veld **Sjablooncode** .</span><span class="sxs-lookup"><span data-stu-id="ec7bc-122">Select a value in the **Template Code** field.</span></span>  
<span data-ttu-id="ec7bc-123">De sjablooncode is de code voor de sjabloon die op de pagina **Sjablonen voor configuratie** is gedefinieerd.</span><span class="sxs-lookup"><span data-stu-id="ec7bc-123">The template code is the code for the template that was defined on the **Configuration Templates** page.</span></span> <span data-ttu-id="ec7bc-124">De gedefinieerde sjabloonwaarden worden toegepast op de API.</span><span class="sxs-lookup"><span data-stu-id="ec7bc-124">The template values defined are applied to the API.</span></span> 
5. <span data-ttu-id="ec7bc-125">Geef in het veld **Voorwaarden** op welke sjabloon moet worden toegepast.</span><span class="sxs-lookup"><span data-stu-id="ec7bc-125">In the **Conditions** field, specify which template should be applied.</span></span>  
<span data-ttu-id="ec7bc-126">De gedefinieerde sjabloon wordt toegepast op een nieuwe record die met de API wordt gemaakt als en alleen als aan de voorwaarden die in het veld **Voorwaarden** zijn gedefinieerd, wordt voldaan door de waarden die al zijn gedefinieerd voor het nieuwe exemplaar van de entiteit.</span><span class="sxs-lookup"><span data-stu-id="ec7bc-126">The defined template is applied to a new record created through the API if, and only if, the conditions defined in the **Conditions** field are met by the values already defined for the new instance of the entity.</span></span>

   * <span data-ttu-id="6f06f-117">Aangepast, gebruikt een aangepaste lay-out in het rapport.</span><span class="sxs-lookup"><span data-stu-id="6f06f-117">Custom, uses a custom layout on the report.</span></span>

> [!NOTE]
> <span data-ttu-id="6f06f-121">Als u **RDLC (ingebouwd)** of **Word (ingebouwd)** kiest en een foutbericht krijgt dat het rapport geen lay-out van de opgegeven soort heeft, moet u een andere lay-outoptie kiezen of een aangepaste rapportlay-out maken van het soort dat u wilt gebruiken.</span><span class="sxs-lookup"><span data-stu-id="6f06f-121">If you choose **RDLC (built-in)** or **Word (built-in)** and you get an error message that the report does not have a layout of the specified type, then you must choose another layout option or create a custom report layout of the type that you want to use.</span></span>
